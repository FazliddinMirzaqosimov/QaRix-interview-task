Setup

Clone the repository

```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

Install dependencies

```bash
pnpm install
```

Setup environment variables

Copy .env.example to .env and fill in your values.

```bash
cp .env.template .env
```


Docker

Build & Run

```bash
docker compose up --build -d
```

Stop containers

```bash
docker compose down
```
